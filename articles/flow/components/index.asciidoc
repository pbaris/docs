= Components

<<{articles}/ds/overview#, Vaadin Design System>> includes a set of components that you can use to build your UI.
The components have a server-side Java API in addition to the TypeScript API for client-side development.

xref:{articles}/ds/components#[Browse components, role="button primary water"]

You use the components by creating one and then adding it to a containing layout.
